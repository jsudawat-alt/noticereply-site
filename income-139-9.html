Section 139(9) — Defective Return

A notice under Section 139(9) is issued when the return filed is found to be defective (missing signature, missing annexure, incorrect format). It requires correction within a specified time or the return is treated as invalid.

Why this notice is issued

  • Return filed without required declaration or signature.
  • Supporting schedules or annexures (Audit report, TDS schedules) missing or incomplete.
  • Incorrect or incompatible return format.

Checklist (documents to prepare)

  • Filed return copy (XML/PDF)
  • Signed declaration or corrected signature page
  • Any missing annexures (audit report, Form 16 etc.)
  • Supporting reconciliations (if schedules mismatch)

Step-by-step reply process

  1. Read the notice to confirm exact defect stated (signature, annexure, format).
  2. Gather the missing documents and correct the return (rectify or re-upload corrected XML if required).
  3. If the portal allows correction, upload corrected return / annexures before the due date mentioned in notice.
  4. If manual submission required, prepare covering letter and attach corrected documents then respond as instructed by the notice (e-filing portal or physical submission).
  5. Keep proof of submission (acknowledgement, upload receipts) and note timelines for follow-up.

Practical drafting tips

  • Keep the reply factual—state what was missing and confirm the corrective action taken.
  • Attach a succinct reconciliation or index of annexures for the officer’s convenience.
  • Always preserve submission receipts and reference numbers in your reply.

Sample reply (editable)

To
The Assessing Officer,
[Address]

Subject: Reply to notice under Section 139(9) dated [dd-mm-yyyy] – PAN [●], AY [●]

Respected Sir/Madam,

1. We refer to the notice dated [date] regarding the return filed under Section 139 for AY [●], alleging defect on account of [state defect].

2. The defect arose due to [brief reason]. We have rectified the matter by [describe action — e.g., uploading signed declaration / attaching Annexure X]. Enclosed: Annexure A – Corrected return (XML/PDF), Annexure B – Audit report / Form 16 (if applicable).

3. We request you to kindly consider the above and treat the return as regularized.

Yours faithfully,
[Name / Authorised Signatory]
